Query — latest_videos
SELECT video_id, title, preview_image, user_id, added_date FROM latest_videos WHERE yyyymmdd = '2024-01-15' ORDER BY added_date DESC LIMIT 10

Partitioned by yyyymmdd — a day-bucket pattern preventing unbounded partition growth while keeping chronological queries efficient.
